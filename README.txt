RESOURCE MANAGEMENT MODULE

 	resourceManager web2project module : this module should be used as an add-on module of web2Project Project Management Software (http://www.web2project.net/) 
	It displays a table that lets you know for a project, a task or all projects the occupancy rate of a resource.

	This module is an add-on module of web2Project. Please refer to the add-on module installation instruction
	in the web2Project documentation
